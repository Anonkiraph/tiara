# Anonkiraph – Romantic Website

This is a simple animated romantic webpage made for Tiara.

## Deploy to GitHub Pages

1. Create a new repository on your GitHub named **Anonkiraph**
2. Upload:
   - index.html
   - style.css
   - (optional) README.md
3. Go to:
   **Settings → Pages**
4. Under "Build and Deployment":
   - Source: **Deploy from branch**
   - Branch: `main`
   - Folder: `/root`
5. Save
